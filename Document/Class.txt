- background
1. planet
2. grid
3. ground

- stage
1. int numStage, numCube: 스테이지 번호, 해당 스테이지에 사용될 큐브의 총 수
2. int map[10][10][10]: 빈곳, 고정된 곳, 채워진 곳 등...
3. Cube* cube[10]: 해당 스테이지에 사용될 큐브(총 10개까지 가능)
4. float timeStage: 게임 시간

- cube
1. int numStage, numCube: 스테이지 번호, 해당 스테이지에 사용될 큐브의 번호
2. int type: Cube가 map에 고정됐는지, 이동가능한지
3. bool cube[10][10][10]: 해당 좌표에 Cube가 있는지 없는지

- Key
q: 게임 종료
y: y축 회전
u/U: Grid On/Off

w: 선택한 Cube 바꾸기(왼쪽)
e: 선택한 Cube 바꾸기(오른쪽)
r: 선택한 Cube 고정
t: 선택한 Cube 이동(default 값)

a/z: 선택한 Cube x축 이동
s/x: 선택한 Cube y축 이동
d/c: 선택한 Cube z축 이동
f: 선택한 Cube x축 회전
g: 선택한 Cube y축 회전
h: 선택한 Cube z축 회전
